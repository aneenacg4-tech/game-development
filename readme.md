# 🎮 Developer Landing Page

A modern and responsive landing page built using **HTML** and **Tailwind CSS**. This project showcases a simple developer-themed UI with a navigation bar and a hero section.

## ✨ Features

* Responsive navigation bar with mobile hamburger menu
* Clean hero section with image and content
* Dark theme with vibrant accent colors
* Hover effects for better user interaction
* Fully responsive layout using Tailwind CSS

## 🚀 Getting Started

1. Download or copy the project files.
2. Ensure you have an internet connection (for CDN links).
3. Open `index.html` in any modern browser.

No installation or setup required.

## 📱 Responsive Design

* Desktop: Full navigation menu displayed
* Mobile: Hamburger menu icon appears
* Layout adjusts from horizontal to vertical on smaller screens

## 🎨 Customization

You can easily customize:

* **Logo text** → Replace `L0go` with your brand name
* **Navigation links** → Update menu items as needed
* **Colors** → Modify Tailwind classes (e.g., `bg-orange-500`)
* **Content** → Replace placeholder text and headings
* **Image** → Swap the Unsplash image URL with your own
